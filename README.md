# Siltawi Digital Marketing Website

A premium digital marketing agency website built with Next.js 15, TypeScript, Tailwind CSS v4, shadcn/ui, and Framer Motion. The project showcases Siltawi's services, portfolio, team members, testimonials, and contact information through a modern, responsive, and conversion-focused design.

## Live Demo

<https://siltawi-company-profile.vercel.app/>

## GitHub Repository

<https://github.com/Hayat-123/siltawi-company-profile.git>

---

## Features

* Modern digital agency landing page
* Fully responsive design
* Sticky navigation header
* Hero section with clear call-to-actions
* Services showcase
* Featured case studies / portfolio section
* About company section
* Team member profiles
* Client testimonials
* Contact section
* Professional footer
* Smooth animations with Framer Motion
* Mobile-first user experience

## Installation

Clone the repository:

```bash
git clone <https://github.com/Hayat-123/siltawi-company-profile.git>
```

Navigate into the project:

```bash
cd siltawi-company-profile
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

---

## Production Build

Build the project:

```bash
npm run build
```

Start production server:

```bash
npm start
```
