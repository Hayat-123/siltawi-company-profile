# Project Context & Rules: Siltawi Digital Marketing

## Core Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4 (using CSS variables in @theme)
- **UI Components:** shadcn/ui (Radix Primitives)
- **Animations:** Framer Motion
- **Icons:** Lucide Icons

## Design System & Theme Guidelines
- **Aesthetic:** Ultra-modern, premium agency UI blended with a sophisticated "Habesha Vibe" (Ethiopian luxury look).
- **Primary Accent:** `var(--color-brand-primary)` (#E5A910 - Adey Abeba Gold)
- **Secondary Accent:** `var(--color-brand-secondary)` (#4A1E17 - Deep Saba Maroon / Coffee Coffee)
- **Background Dark:** `var(--color-brand-dark)` (#120F0E - Rich Espresso Charcoal)
- **Background Light:** `var(--color-brand-light)` (#FAFAF7 - Warm Soft Cream)

## Structural Goal
Build a gorgeous, responsive single-page application landing page split into clean, modular components inside `src/components/`. Use standard TypeScript typing, clean layouts, and smooth transition animations.
